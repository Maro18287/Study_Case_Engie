# Study_Case_Engie

Repository Structure
notebooks/
This folder contains three Jupyter notebooks:
processing_and_features.ipynb
This notebook contains the material used to answer Questions 1 and 2, as well as part of Question 3.
It includes data cleaning, feature creation, and the integration of the different data sources.
model.ipynb
This is where the model is trained, and where you will find the results, evaluation metrics, and related analyses.
bess.ipynb
This notebook answers Question 5.
It includes the construction of the BESS strategy and the computation of the PnL.

Data Folder
data/ contains all datasets used in the project:
raw/ — the original unprocessed data.
processed_data/ — the cleaned and transformed datasets used for modeling.
A CSV file containing the model predictions is also included in this folder.
