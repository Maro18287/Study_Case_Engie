# Study_Case_Engie

Repository Structure

notebooks/
- processing_and_features.ipynb : contains the work used to answer Questions 1 and 2, and part of Question 3. It includes data cleaning, feature engineering, and the integration of the different data sources.
- model.ipynb : contains the model training, results, evaluation metrics, and related analyses.
- bess.ipynb : answers Question 5, including the construction of the BESS strategy and the PnL calculation.

data/
- raw/ : original unprocessed data.
- processed_data/ : cleaned and transformed datasets used for modeling.
- A CSV file with the model predictions is also included in this folder.
